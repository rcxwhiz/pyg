
        Welcome to the PYG automatic Python grader
                     By Josh Bedwell
-----------------------------------------------------------

               You can use this program to
- Run key files against test cases and generate output files
- Grade graph outputs using image recognition
- Have assignments that have multiple parts (ex. 1, 2a, B)
- Create a ruberic to weight the parts
- Run a batch of student code against the key output files
- Automatically generate an Excel report with all the results
from the student files
- View the actual outputs of student code from a report
- Have a copy past assister for entering grades
